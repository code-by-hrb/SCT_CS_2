🛡️ SCT_CS_2: VaultGuard

Ultra-private, client-side pixel cryptography studio. Secure sensitive visual assets locally using mathematical ciphers and color channel manipulation directly in your browser.


💼 SkillCraft Technology Internship Project | Cyber Security Task #2


🎯 Main Goal

To provide a 100% private, serverless environment for securing images locally before they are shared online. By executing all cryptographic formulas entirely within the browser via the HTML5 Canvas API, VaultGuard eliminates risks of network interception and server-side leaks.

🔥 Core Features

   ✅ Dual Ciphers: Toggle between a Symmetric XOR Stream Cipher or RGB Channel Swap Protocol driven by user-defined numerical keys (1–255).
   
   ✅ Total Privacy: Pure client-side execution. Images never leave your device, and zero data hits the network.
   
   ✅ Responsive Cyberpunk UI: A sleek, neon-tinted dark workspace optimized for desktop monitors and mobile layouts.
   
   ✅ Instant Preview: Renders the scrambled matrix immediately with an on-demand download button.
    

🛠️ Tech Stack

    HTML5: Structuring the core application workspace components.
    CSS3: Implementing full responsive layout grids, responsive media queries, and dark-theme aesthetics.
    JavaScript (ES6): Executing raw pixel data manipulation algorithms and image buffer processing.

🚀 Quick Start

    1. Clone this repository.
    2. Open index.html in any web 
    3. Choose a key, select a protocol, drop an image, and secure your file!
